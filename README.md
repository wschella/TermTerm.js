# TermTerm.js

 Small utility that converts string representations of RDF terms to a JSON representation as specified by RDF.js

## Setup

1. Install NPM
1. `npm install -g typescript`
1. `npm install -g gulp-cli`
1. `npm install`
1. `npm test`
